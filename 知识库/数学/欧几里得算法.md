## 1. 核心定义
欧几里得算法（辗转相除法）是用于计算两个非负整数 $a, b$ 的**最大公约数**的高效算法。
核心原理基于以下性质：
$$(a, b) = (b, a \bmod b)$$

## 2. 算法流程
设 $r_0 = a, r_1 = b$ (假设 $a \ge b > 0$)。
进行连续带余除法：
$$
\begin{aligned}
r_0 &= r_1 q_1 + r_2, & 0 < r_2 < r_1 \\
r_1 &= r_2 q_2 + r_3, & 0 < r_3 < r_2 \\
&\vdots \\
r_{k-2} &= r_{k-1} q_{k-1} + r_k, & 0 < r_k < r_{k-1} \\
r_{k-1} &= r_k q_k + 0 & (\text{整除，余数为0})
\end{aligned}
$$
则最后一个**非零余数** $r_k$ 即为最大公约数 $(a, b)$。

```python
# 核心：gcd(a,b) == gcd(b,a mod b)
def gcd(a,b):
	while b != 0:
		a,b = b,a % b
	return a
```

## 3. 示例
计算 $(198, 252)$：
1. $252 = 1 \times 198 + 54$
2. $198 = 3 \times 54 + 36$
3. $54 = 1 \times 36 + 18$
4. $36 = 2 \times 18 + 0$
$\therefore (198, 252) = 18$
