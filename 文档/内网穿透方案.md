> [!summary] 核心目标
> 将内网服务（如 Web、SSH、NAS）暴露到公网，或实现不同网络环境下的设备互联。

## 方案对比

| 方案               | 核心技术             | 适用场景     | 依赖条件          |
| :--------------- | :--------------- | :------- | :------------ |
| [[Nginx 反向代理]]   | HTTP 反代          | 建站、域名访问  | 需公网 IP        |
| [[Autossh 隧道]]   | SSH 隧道           | 临时远程管理   | 需公网 IP、SSH 服务 |
| [[Tailscale 组网]] | Mesh VPN         | 多设备互联、管理 | 无需公网 IP       |
| [[Frp 内网穿透]]     | Client-Server 转发 | 高性能暴露服务  | 需公网 IP (中转)   |
